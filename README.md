# アプリケーション名

Photo-Search

# URL

デプロイしていません

# アプリケーション概要

reactで作成した写真検索のSPAです。
[unsplash](https://unsplash.com/)及び[pixabay](https://pixabay.com/ja/)のAPIを用いています。
react-iframeを利用して[kaboompics](https://kaboompics.com/), [Negative Space](https://negativespace.co/)のwebサイトも埋め込んであります。

# 利用方法
1. 検索窓に検索したい単語を入力
2. "search..."をクリック
