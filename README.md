# アプリケーション名

Photo-Search

# URL

デプロイしていません

# アプリケーション概要

reactで作成した写真検索のSPAです。
[unsplash](https://unsplash.com/)のAPIを用いています。

# 利用方法
1. 検索窓に検索したい単語を入力
2. "search..."をクリック
